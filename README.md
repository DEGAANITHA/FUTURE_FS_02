📊 Mini CRM - Client Lead Management System

📌 Project Overview

The Mini CRM (Client Lead Management System) is a web application designed to manage client leads generated from website contact forms. It allows users to add, view, and manage leads efficiently in a simple and user-friendly interface.

---

🚀 Features

- ➕ Add new client leads
- 📋 View all leads in a structured table
- 🔄 Update lead status (New, Contacted, Converted)
- 📝 Manage lead information (Name, Email, Source)
- ⚡ Fast and responsive UI

---

🛠️ Technologies Used

Frontend:

- HTML
- CSS
- JavaScript / React

Backend:

- Node.js
- Express.js

Database:

- MongoDB / MySQL

---

📂 Project Structure

mini-crm/
 ├── frontend/   # User Interface
 └── backend/    # Server & APIs

---

▶️ How to Run the Project

1. Clone the Repository

git clone https://github.com/DEGAANITHA/FUTURE_FS_02.git

2. Run Frontend

cd frontend
npm install
npm start

3. Run Backend

cd backend
npm install
node server.js
